Simple Chat Client

Simple Chat Client is a lightweight and simple program which allows talking in the czat.onet.pl without using a browser or java.

The program of the assumptions is small, lightweight and stable client - so will never have such things as scripts or using a speech synthesizer.

The program is written in QT, making it a multi-Platform - works on both Windows and Linux.

Available under GPL v3 in binary (windows) as well as in source code.
